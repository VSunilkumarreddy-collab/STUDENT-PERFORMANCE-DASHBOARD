# STUDENT-PERFORMANCE-DASHBOARD



An interactive Power BI dashboard to analyze student performance across subjects, attendance, and classes.

---

## Files

| File | Description |
|------|-------------|
| `student_performance.csv` | Dataset |

---

## How to Run

1. Download **Power BI Desktop** — free at microsoft.com/power-bi
2. Load `student_performance.csv` via **Home → Get Data → Text/CSV**
3. Build all visuals following the dashboard structure
4. Save as `.pbix` and export as PDF

---

## Dashboard Includes

- 3 KPI Cards — Average Marks, Attendance, Total Students
- Bar Chart — Average Marks by Subject
- Bar Chart — Top 10 Student Rankings
- Scatter Chart — Attendance vs Marks
- Histogram — Marks Distribution
- 4 Slicers — Class, Subject, Gender, Teacher
